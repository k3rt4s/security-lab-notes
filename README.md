## Usage notes

- All material is authorized lab, training, or defensive-assessment reference only.
- Personal names, company names, customer domains, credentials, and internal environment details are kept out of committed notes. Tokenized placeholders (`<lab-ip>`, `<lab-password>`, `example.test`) are used throughout.
- Prefer maintained upstream documentation for tool installation. This repository is for notes, checklists, and review context, not maintained automation.
